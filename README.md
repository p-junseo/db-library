# 2025-1 데이터베이스 기말 프로젝트 : 라이브러리
데이터베이스를 활용한 온라인 도서관 프로그램 제작

## 개발 환경
- C#, MySQL, Naver API 등
- 콘솔 조작으로 사용 가능

## 지원 기능

### 공통 기능
- 로그인
- 로그아웃
- 관내 보유 도서 검색
- 네이버 도서 검색
- 요청 도서 조회

### 유저 모드 기능
- 회원 가입
- 도서 대출
- 도서 반납
- 대출 내역 조회
- 반납 내역 조회
- 내 정보 수정
- 회원 탈퇴
- 도서 요청

### 매니저 모드 기능
- 도서 추가
- 도서 수정
- 도서 삭제
- 반납 내역 조회
- 요청 도서 추가
- 유저 삭제
- 로그 조회
- 로그 삭제
- 로그 파일 저장
- 로그 파일 삭제
